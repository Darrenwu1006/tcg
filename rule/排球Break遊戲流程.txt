排球Break遊戲流程


🏐 遊戲回合與階段順序（Phase Sequence）
I. 遊戲準備階段（Game Setup）
在遊戲正式開始前，雙方玩家必須完成以下步驟：
1. 提示牌組（デッキの提示）： 雙方提示各自的牌組（須滿足 40 張且事件卡 ≤8 張的構築條件）。
2. 決定發球權（サーブ権の決定）： 隨機選擇一名玩家，該玩家決定是要擁有首次發球權，還是放棄發球權（由另一位玩家擁有）。
3. 牌組洗牌與放置（デッキのシャッフル）： 雙方玩家洗牌後，將牌組背面朝上放置於牌組區（デッキエリア）。
4. 抽起始手牌（最初の手札）： 雙方玩家從牌組抽取 6 張卡作為起始手牌。
5. 調整手牌（引き直し）： 從擁有首次發球權的玩家開始，雙方可進行一次換牌（將任意張數卡片放回牌組洗牌，然後補滿至 6 張）。
6. 配置 Set 卡（セットカードの配置）： 雙方玩家從牌組頂端抽取 2 張卡，背面朝下放置於 Set 區（セットエリア）。
7. 遊戲開始： 擁有發球權的玩家進行「發球階段」（サーブフェイズ）。

________________


II. 回合類型與階段結構
一場遊戲中的回合（ターン）分為三種基本構成：
1. Set 的第一回合：發球回合
階段 (Phase)
	步驟 (Step)
	內容與說明
	來源
	發球階段 (Serve Phase)
	1. 開始
	檢查發動條件為「開始時」的技能。執行檢查程序 (Check Process)。
	

	

	2. 登場步驟 (登場ステップ)
	玩家必須登場 1 張角色卡到「發球區」。若未登場，則宣告 Lost。
	

	

	3. 自由步驟 (フリーステップ)
	玩家可使用「主動型技能」或「事件卡」。若宣告 Lost，則進入 Lost Set 步驟。
	

	

	4. 攻擊點數計算步驟 (OP 算出)
	計算發球角色的發球點數（Serve Point）作為攻擊點數 (Offense Point, OP)。
	

	

	5. 結束
	結束該階段效果 → 進入結束階段 (End Phase)。
	

	________________


2. 後續回合 (非第一回合)：選擇防守模式
在第一回合之後，回合開始於 開始階段（Start Phase）。
階段 (Phase)
	步驟 (Step)
	內容與說明
	來源
	開始階段 (Start Phase)
	1. 開始
	檢查發動條件為「開始時」的技能。
	

	

	2. 選擇防守
	玩家必須宣告本回合是要進行 攔網 (Block Phase) 還是 接球 (Receive Phase)。
	

	

	3. 結束
	結束該階段效果 → 進入選擇的階段。
	

	--------------------------------------------------------------------------------
________________


III. 攔網軸（Block Path）
如果玩家選擇了 攔網（Block），回合將依序進行以下階段：
階段 (Phase)
	步驟 (Step)
	內容與說明
	來源
	攔網階段 (Block Phase)
	1. 開始
	檢查發動條件為「開始時」的技能。
	

	

	2. 登場步驟 (登場ステップ)
	玩家可登場 1 至 3 張攔網角色到「攔網區」。若未登場，則宣告 Lost。
	

	

	3. 自由步驟 (フリーステップ)
	玩家可使用技能或事件卡。
	

	

	4. 防守點數計算步驟 (DP 算出)
	計算攔網角色的 Block Point 總和作為防守點數 (Defense Point, DP)。
	

	

	5. 判定步驟 (Judge Step)
	比較 DP 與對手的 OP。若 DP < OP，則攔網失敗，宣告 Lost。
	

	

	6. 攻擊點數計算步驟 (OP 算出)
	計算本次攔網成功後的 OP（通常為 0 點，除非觸發黑尾的 ドシャット(5) 等技能）。
	

	

	7. 結束
	側邊攔網者（サイドブロッカー）移動到棄牌區（ドロップエリア）。 → 進入結束階段。
	

	--------------------------------------------------------------------------------
________________


IV. 接球軸（Receive Path）
如果玩家選擇了 接球（Receive），回合將依序進行以下階段：
階段 (Phase)
	步驟 (Step)
	內容與說明
	來源
	抽牌階段 (Draw Phase)
	1. 開始
	檢查發動條件為「開始時」的技能。
	

	

	2. 抽牌步驟 (ドローステップ)
	玩家從牌組抽 1 張卡（即俗稱的 レシーブドロー）。
	

	

	3. 自由步驟 (フリーステップ)
	玩家可使用技能或事件卡。
	

	

	4. 結束
	結束該階段效果 → 進入接球階段。
	

	接球階段 (Receive Phase)
	1. 開始
	檢查發動條件為「開始時」的技能。
	

	

	2. 登場步驟 (登場ステップ)
	玩家必須登場 1 張接球角色到「接球區」。若未登場，則宣告 Lost。
	

	

	3. 自由步驟 (フリーステップ)
	玩家可使用技能或事件卡。
	

	

	4. 防守點數計算步驟 (DP 算出)
	計算接球角色的 Receive Point 作為 DP。
	

	

	5. 判定步驟 (Judge Step)
	比較 DP 與對手的 OP。若 DP < OP，則接球失敗，宣告 Lost。
	

	

	6. 結束
	結束該階段效果 → 進入托球階段。
	

	托球階段 (Toss Phase)
	1. 開始
	檢查發動條件為「開始時」的技能。
	

	

	2. 登場步驟 (登場ステップ)
	玩家必須登場 1 張托球角色到「托球區」。注意：不可與「接球角色」同名。若未登場，則宣告 Lost。
	

	

	3. 自由步驟 (フリーステップ)
	玩家可使用技能或事件卡。
	

	

	4. 結束
	結束該階段效果 → 進入攻擊階段。
	

	攻擊階段 (Attack Phase)
	1. 開始
	檢查發動條件為「開始時」的技能。
	

	

	2. 登場步驟 (登場ステップ)
	玩家必須登場 1 張攻擊角色到「攻擊區」。注意：不可與「托球角色」同名。若未登場，則宣告 Lost。
	

	

	3. 自由步驟 (フリーステップ)
	玩家可使用技能或事件卡。
	

	

	4. 攻擊點數計算步驟 (OP 算出)
	計算 Toss Point（托球角色） + Attack Point（攻擊角色）的總和作為 OP。
	

	

	5. 結束
	結束該階段效果 → 進入結束階段。
	

	--------------------------------------------------------------------------------
________________


V. 回合最終階段（End of Turn）
所有回合（無論是發球軸、攔網軸或接球軸）的最後一個階段都是 結束階段（End Phase）。
階段 (Phase)
	步驟 (Step)
	內容與說明
	來源
	結束階段 (End Phase)
	1. 開始
	檢查發動條件為「開始時」或「回合結束時」的技能。
	

	

	2. 清理步驟 (Cleanup Step)
	適用於該回合的持續效果（持續效果、數值修正或固定效果）結束。
	

	

	3. 結束
	回合結束，將行動權交給對手（非回合玩家成為新的回合玩家），進入新的開始階段。
	

	--------------------------------------------------------------------------------
________________


VI. Lost 與 Interval 流程
當一方玩家宣告 Lost 時，遊戲進程會立即中斷，並執行以下流程：
1. Lost Set 步驟 (ロストセットステップ)：
    ◦ 攻擊點數（OP）和防守點數（DP）立即消滅。
    ◦ 檢查發動條件為「宣告 Lost 時」的技能。
    ◦ Set 中持續的效果和待機狀態的技能立即消失。
    ◦ 步驟結束 → 進入 Interval。
2. Interval (インターバル)：
    ◦ 雙方玩家將手牌補滿至 6 張。
    ◦ 宣告 Lost 的玩家，將 Set 區的 1 張 Set 卡加入手牌。
    ◦ Set 中持續的效果結束。
    ◦ 未 Lost 的玩家獲得下一 Set 的發球權，進入下一個 Set 的發球階段。
如果玩家的 Set 區卡片耗盡（只剩 0 張）時宣告 Lost，則該玩家立即判定敗北，遊戲結束。
